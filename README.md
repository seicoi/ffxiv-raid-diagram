# FFXIV Raid Diagram

FFXIV向けのレイド・ギミック解説図作成ツールです。

フィールド画像、プレイヤー・ボスアイコン、マーカー、AoE画像、バフ・デバフなどの画像アセットをインポートし、キャンバス上に自由に配置できます。

また、ガイド線を利用してフィールドサイズを定義することで、ゲーム内に近いスケールでAoEや配置を再現できます。

## 主な機能

* 各種画像アセットのインポート
* フィールド画像の設定
* プレイヤーアイコン配置
* ボスアイコン配置
* 円形・矩形・扇形・ドーナツAoE作成
* 画像AoEの配置
* マーカー配置
* バフ・デバフ表示
* レイヤー管理
* グリッド表示
* Undo / Redo
* PNG出力
* JSON保存 / 読み込み
* IndexedDBによるブラウザ内保存

## 特徴

### 自由なアセット管理

フィールド画像だけでなく、プレイヤー・ボスアイコン、マーカー、AoE画像、バフ・デバフなど、任意の画像をアセットとしてインポートできます。

用途に応じてカテゴリ分けしながら管理でき、キャンバスへドラッグ＆ドロップで配置できます。

### 正確なスケール設定

ガイド線をフィールドに合わせてサイズを設定することで、AoEやオブジェクトをメートル単位で配置できます。

攻略図としての見やすさだけでなく、距離感やAoEサイズの検証にも利用できます。

### 柔軟なオブジェクト管理

プレイヤー、ボス、AoE、マーカー、バフ・デバフを個別に管理できます。

レイヤー機能により表示順の変更や表示/非表示の切り替えも可能です。

## 保存

以下の保存方法に対応しています。

* 自動保存（IndexedDB）
* JSON保存 / 読み込み
* PNG出力

## 利用例

* レイド攻略図の作成
* 固定向け資料の作成
* 動画用解説素材の作成
* ギミック整理
* 配置検証
* AoEサイズ確認
